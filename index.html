PHP Full Stack App

Intro

The application for requests/tickets/tasks management raised from the need of ecsaping from shared excel files where the risk of getting things messy is very very high. That affects the data quality which has enormous imapct on further reporting and the analysis.

  • Getting feedback for a consumer/client is the best way to deploy the process continues improvement.
  • Giving communication channel requester can post a feedback or question in much easier way.
  • Application allows to comment each post and tracks posting time.

Features

App includes following features:

  • Relational database model.
  • Register/Login/Logout system.
  • Form data against database validation.
  • AJAX applied for geting data without refreshing page.
